# REASON - Beginner Guide

This guide is for first-time players.  
If you follow it step by step, you can start playing in 2-3 minutes.

## 1) What is the goal?

Your goal is simple:
- Survive enemy waves.
- Beat bosses.
- Buy upgrades between waves.
- Build a stronger run and push to higher waves.

Core loop:
1. Fight a wave.
2. Enter wave pause.
3. Buy upgrades/items in shops.
4. Confirm next wave.
5. Repeat.

## 2) Quick Start (First Run)

1. Press `Start Run`.
2. Move with `WASD`.
3. Aim with mouse.
4. Hold left mouse (or `Space`) to shoot.
5. Use abilities with `Z`, `X`, `C`.
6. When wave ends, open shop and buy power.
7. Press `Confirm / Next Wave`.

If you die, spend Meta Credits to unlock stronger permanent options.

## 3) Controls

Default keyboard + mouse:
- Move: `W`, `A`, `S`, `D`
- Aim: mouse
- Shoot: mouse / `Space`
- Abilities: `Z` (Dash), `X` (Shield), `C` (EMP)
- Item slots: `Q`, `E`, `R`
- Weapon slots: `1`, `2`, `3`
- Confirm next wave: `Enter`
- Open shop (during wave pause): `B`
- Pause overlay: `Esc`
- Quick Save: top `Save Now` button (or bound key)

Mobile:
- Use virtual joystick for movement.
- Use on-screen arcade buttons for fire, abilities, items, and weapon swap.

You can rebind keys in `Settings -> Open Control Deck`.

## 4) Combat Basics

- Keep moving. Standing still usually kills runs.
- Shield (`X`) is best when incoming damage is high.
- EMP (`C`) is best when enemies are grouped.
- Dash (`Z`) is best for escaping danger or repositioning.
- Swap weapons (`1/2/3`) based on enemy type and distance.

## 5) Waves and Shops

When a wave ends:
- Time pauses.
- You can open:
  - Weapon Shop
  - Item Shop
  - Meta Shop
  - Exchange (casino machine)

Recommended order for beginners:
1. Buy damage or fire rate.
2. Buy survivability (HP/shield/defense).
3. Buy utility item if affordable.
4. Confirm next wave.

## 6) Progression Systems

Run resources:
- `R` (Resources): used inside the current run.
- Lost when run ends.

Meta currency:
- `MC` (Meta Credits): permanent progression currency.
- Used for unlocks in Meta systems.

Meta Tree:
- Permanent bonuses like damage, cooldown, survivability, economy, combo scaling.
- Spend `MC` to unlock nodes.

## 7) Save and Load

- `Save Now` (top bar) does a quick save.
- Slot 1 is reserved for quick-save overwrite.
- Use save slots UI for manual backup saves (recommended: slots 2+).
- `Continue` restores the latest run snapshot when available.

## 8) Boss Fights

Bosses appear on boss waves.

During boss fights:
- Watch cast bars and telegraphs.
- Do not tunnel damage all the time.
- Prioritize survival and spacing first, damage second.

If intro appears, you can skip it with click/tap/`Space`.

## 9) Exchange Machine (MC -> R)

The Exchange shop uses a slot machine:
- Set bet amount (MC).
- Pull lever to spin.
- Winning combinations return `R`.

Important:
- You spend `MC` to play.
- Use it as risk/reward economy, not guaranteed profit.

## 10) Beginner Build Tips

Good early priorities:
1. Stable damage increase.
2. Cooldown reduction.
3. One defensive tool (shield or sustain item).
4. One crowd-control option (EMP or AoE weapon).

If stuck:
- Lower difficulty.
- Focus on movement + wave shop timing.
- Avoid buying too many low-impact upgrades.

## 12) Troubleshooting

If controls feel wrong:
- Open keybind menu and reset/rebind.

If UI is crowded on mobile:
- Use HUD editor and adjust joystick/action positions.

If performance drops:
- Set VFX quality to `Medium` or `Low`.

If progress looks missing:
- Check correct save slot.
- Use `Continue` for run snapshots.
